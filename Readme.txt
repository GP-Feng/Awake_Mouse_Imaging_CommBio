Readme:
Resources and data processing codes for the paper: High-resolution structural and functional retinal imaging in the awake behaving mouse
Directories:
*.mp4: Videos instruct to assemble the head-restrained mouse setup for awake, behaving mouse retinal imaging.
\Awake_mouse_setup: Assembly instruction and 3D printing files for the head-restrained mouse setup for awake, behaving mouse retinal imaging.
\ AOSLO_linescanned_tracking: Code used to track the high-frequency eye motions from linescanned AOSLO data
\Pupil_tracking: Code used to track and segment mouse pupil
\Retina_tracking_SLO: Code used to track retinal motions from SLO videos
\OCT_segmentation: Code used to detect and segment mouse retinal layers for TRT measurement
